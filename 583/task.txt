Прочитайте код скрипта на Python и напишите запрос, результат которого будет совпадать с результатом выполнения скрипта.

Скрипт и файл базы данных можно увидеть в терминале в каталоге /tmp:

cd /tmp
sqlite3 ./m3_hw_task2.db "SELECT * FROM Conference"
# выход из просмотрщика less осуществляется нажатием клавиши q
less m3_hw_task2.py

или скачать из материалов курса: m3_hw_task2.py m3_hw_task2.db и гонять на своей машине.

Ответом на задание будет текст запроса, записанный в файл /tmp/answer, без кавычек, с плейсхолдером ? на месте, где запрос ожидает параметр (если он вообще его будет ожидать). Пример валидного ответа:


SELECT * FROM Conference WHERE name=?
